# Contexto

Aplicativo Flutter para organizar screenshots pelo significado. A proposta do
MVP é funcionar localmente, offline e sem backend ou serviços pagos.

## Requisitos

- Flutter compatível com Dart 3.12 ou superior
- Android SDK e um dispositivo ou emulador Android configurado

## Executar o projeto

```bash
flutter pub get
flutter run
```

## Verificações

```bash
dart format .
flutter analyze
flutter test
```
